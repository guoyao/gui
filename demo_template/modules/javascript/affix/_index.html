Affix affix.js

Example

The message dialog on the bottom right is a live demo of the affix plugin.

Usage

Via data attributes

To easily add affix behavior to any element, just add class="js-affix" to the element. Then use offsets to define position relative to the document.

<div class="js-affix" data-offset-top="200px">...</div>

Via JavaScript

Call the affix plugin via JavaScript:

$('.js-affix').guiAffix({
    offset: {
      bottom: "10px",
      right: "10px"
    }
  })

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-offset-top="200".

Name type description
offset number | object Pixels to offset from screen when calculating position of scroll. If a single number is provided, the offset will be applied in both top and left directions. To provide a unique, bottom and right offset just provide an object offset: { bottom: "10px" } or offset: { bottom: "10px", right: "10px" }.
